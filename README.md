## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/madx900/linux-tutorial.git
   cd linux-tutorial
   ```

2. Navigate through the documentation:
   - Start with the [Introduction to Linux](docs/en/basics/01-introduction.md)
   - Follow the sections in order for a structured learning path
   - Use the search function to find specific topics

   - Understanding Linux basics
   - Command line navigation

   - User management

   - Networking basics
   - Shell scripting fundamentals
   - Service management

3. **Advanced Level** (Weeks 9-12)
   - Container management
   - Security hardening
   - Performance optimization
   - Automation and DevOps practices
